To-Do App

A simple and intuitive **To-Do application** to help users organize tasks efficiently. Built with [ HTML, CSS, JavaScript ].

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Edit existing tasks
- Responsive design for mobile and desktop
- [Optional] Filter tasks (All / Completed / Pending)
-Dark mode feature

Usage

Enter a task in the input field.

Click the Add button or press Enter.

Click the added tasks to mark as completed.

Click the Edit button to modify a task.

Click the Delete button to remove a task.

Technologies Used

HTML5

CSS3

JavaScript (or React, Vue, etc.)
